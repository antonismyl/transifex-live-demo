Introduction

Lumen's JavaScript SDK lets you send events from any web browser with a single script tag. It's designed to be small (under 8 KB gzipped), robust to network failures, and easy to integrate with the rest of your stack.

This page is also a working example of a Transifex Live integration — the site uses a custom language picker and several patterns for marking translatable content.

Install the SDK

Add the snippet to your site's <head>. Replace YOUR_PROJECT_KEY with the key from your project settings.

<script>
  (function(l,u,m,e,n){l.lumen=l.lumen||function(){(l.lumen.q=l.lumen.q||[]).push(arguments)};
  var s=u.createElement(m);s.async=1;s.src=e;var f=u.getElementsByTagName(m)[0];
  f.parentNode.insertBefore(s,f);})(window,document,'script','https://cdn.lumen.io/v1.js');
  lumen('init', 'YOUR_PROJECT_KEY');
</script>

Send your first event

Once the SDK is initialized, any call to lumen('track', …) is buffered until the script loads, then flushed in order.

lumen('track', 'signup_completed', {
  plan: 'pro',
  referrer: document.referrer
});

track(eventName, properties)

Records an event for the current user.

ParameterTypeNotes
eventNamestringRequired. Use snake_case. Max 64 characters.
propertiesobjectOptional. Values must be JSON-serializable.

identify(userId, traits)

Associates subsequent events with a known user.

ParameterTypeNotes
userIdstringRequired. Your internal user identifier.
traitsobjectOptional. User-level properties merged with existing traits.

REST: creating a cohort

The REST API accepts authenticated requests with a bearer token. Cohorts created here appear in the Lumen UI within a few seconds.

curl -X POST https://api.lumen.io/v2/cohorts \
  -H "Authorization: Bearer $LUMEN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Weekly active",
    "definition": {
      "all_of": [
        { "event": "session_started", "count": { "gte": 3 }, "within": "7d" }
      ]
    }
  }'

Next steps

To experiment with the Transifex Live integration on your own copy of this demo, swap the placeholder api_key on every page for a real key from your Transifex Live project. Once the snippet loads, the language picker in the top-right activates and page content translates in place — no page reload required.
